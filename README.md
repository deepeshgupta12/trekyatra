# TrekYatra Monorepo

This repository contains the end-to-end TrekYatra product:
- `apps/web-static` — uploaded static frontend source-of-truth from design export
- `services/api` — FastAPI backend (to be created stepwise)
- `docs` — process, tracker, dependency map, and step execution docs
- `scripts` — helper scripts for setup, validation, and Nexus/GitNexus workflows

Refer to the docs in `docs/` before every implementation step.
